# Настройка и установка

## Подготовка

Надо установить node (лучше 14 версию, так как я на нем сделал).

[Установка node](https://nodejs.org/en/)

### Установить зависимости
```
npm install
```

### Запустить сервер
```
node index.js
```

проверить [http://localhost:3000/](http://localhost:3000/)
